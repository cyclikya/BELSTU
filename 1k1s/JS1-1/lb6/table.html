Свойство Значение
Navigator.userAgent (Информация о браузере пользователя)
Navigator.appVersion(версия браузера)
Navigator.appName(название браузера)
Navigator.appCodeName(кодовое название браузера)
Navigator.platform(ОС)
Navigator.javaEnabled()(включена ли поддержка Java в браузере пользователя )
Screen.width / Screen.height(определяет ширину/высоту экрана пользователя)
Screen.colorDepth(узнать глубину цвета (измеряется в битах на пиксель))
HistoryLength(узнать количество посещенных URL, хранящихся в списке)
Location.href(хранит URL текущего документа целиком)
Location.pathname(узнать путь к загруженному документу)
Location.host(содержит имя домена загруженного документа)
