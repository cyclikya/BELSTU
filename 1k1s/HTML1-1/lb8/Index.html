Информация о свойствах CSS

CSS (Cascading Style Sheets) - это язык стилей, который определяет внешний вид HTML-документов...

1. Текстовое оформление

  • font-family: Задает шрифт текста.
  • font-size: Определяет размер шрифта.
  • color: Устанавливает цвет текста.

2. Размещение и выравнивание

  • margin и padding: Управляют внешними и внутренними отступами соответственно.
  • text-align: Выравнивает текст по горизонтали.

3. Рамки и фоны

  • border: Определяет стиль, ширину и цвет границы элемента.
  • background-color: Задает цвет фона элемента.

4. Позиционирование

  • position: Определяет метод позиционирования элемента.
  • top, right, bottom, left: Используются для точной настройки позиции элемента.

5. Флексбоксы и сетки

  • display: flex: Создает контейнер с гибкими элементами внутри.
  • grid-template-columns, grid-template-rows: Задают сетку для элементов.
