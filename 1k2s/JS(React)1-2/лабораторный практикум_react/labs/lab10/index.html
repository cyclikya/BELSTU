Лабораторная работа №10

Для выполнения этой лабораторной работы вам понадобится компонент Catalog, разработанный в лабораторной работе №8.

Задание 1

Модифицируйте компонент каталога товаров Catalog, добавив каждому товару информацию о массе.


Реализуйте компонент OrderForm — многостраничный диалог оформления заказа в интернет-магазине:

1 страница:

  • Список товаров, добавленных в корзину. У каждого товара есть:
    • чекбокс, при помощи которого товар можно не добавлять в заказ;
    • поле для ввода количества товаров, которые клиент хочет заказать (проверьте, есть ли столько товаров на складе);
    • кнопка для удаления из корзины.


2 страница:

  • Радиокнопки для выбора способа доставки:
    • курьером (если стоимость заказа меньше 200 рублей, то стоимость 10 рублей, если больше, то бесплатно),
    • почтой (5 рублей за каждый килограмм массы),
    • самовывоз (бесплатно).
  • Текстовое поле для указания адреса доставки. Неактивно, если выбран самовывоз.
  • Радиокнопки для выбора способа оплаты:
    • наличными,
    • банковской картой,
    • банковским переводом.
  • Кнопки для перехода к следующей и предыдущей страницам.

Для работы с деревом состояния используйте хранилище Redux.
