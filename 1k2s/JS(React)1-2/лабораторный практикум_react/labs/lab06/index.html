Лабораторная работа №6
Задание 1

Создайте компонент ColorPicker, который будет представлять выбор цвета с использованием хуков в React.

  • Создайте компонент ColorPicker, который будет отображать палитру цветов.

  • Используйте хук useState для создания состояния selectedColor, которое будет хранить выбранный цвет.

  • Отобразите палитру цветов на странице. Каждый цвет должен быть представлен в виде прямоугольника или круга.

  • Добавьте обработчик событий, чтобы при выборе цвета он сохранялся в состоянии selectedColor.

  • Отобразите выбранный цвет на странице, например, с помощью текста или прямоугольника с выбранным цветом внутри.

  • Добавьте возможность пользователю изменять выбранный цвет. Например, можно добавить кнопку Reset, при нажатии на которую выбранный цвет будет сброшен на начальное значение.

  • Добавьте возможность пользователю сохранить выбранный цвет. Например, можно добавить кнопку Save, при нажатии на которую выбранный цвет будет добавлен в список сохраненных цветов.

  • Используйте хук useEffect для сохранения списка сохраненных цветов в локальном хранилище браузера. При загрузке страницы проверьте, есть ли сохраненные цвета в хранилище, и если есть, отобразите их на странице.
