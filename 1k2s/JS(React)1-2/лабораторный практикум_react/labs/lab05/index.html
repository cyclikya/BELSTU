Задание 1

Создайте компонент ContactForm, который будет представлять форму для ввода информации о контакте. Форма должна содержать поля для ввода имени, электронной почты и сообщения, а также кнопку Submit.

  • Создайте компонент ContactForm, который будет отображать форму.

  • Добавьте поле ввода для имени контакта. Пользователь должен иметь возможность ввести имя в это поле.

  • Добавьте поле ввода для электронной почты. Пользователь должен иметь возможность ввести свой адрес электронной почты.

  • Добавьте поле ввода для сообщения. Пользователь должен иметь возможность ввести текст сообщения.

  • Пол пользователя при помощи радиокнопок.

  • Реализуйте валидацию полей формы. Убедитесь, что имя, электронная почта и сообщение были введены корректно перед отправкой формы. Если какое-либо поле не прошло валидацию, выведите сообщение об ошибке пользователю.

  • Добавьте функциональность очистки полей формы после отправки. После успешной отправки формы все поля должны быть очищены, чтобы пользователь мог ввести новую информацию.

  • Добавьте кнопку Submit. При нажатии на эту кнопку, данные из формы должны быть отображены в таблице на вашей странице.
