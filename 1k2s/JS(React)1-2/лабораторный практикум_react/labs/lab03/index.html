Лабораторная работа №3
Задание 1

Создайте компонент EmailForm. Компонент должен предоставлять форму для ввода электронной почты и кнопку "Отправить". При отправке формы, компонент должен валидировать введенный адрес электронной почты и выводить сообщение об успешной отправке или об ошибке валидации.

Задание 2

Реализуйте компонент ProductCatalog для отображения каталога товаров с возможностью сортировки по различным значениям. Таблица должна содержать следующие столбцы: номер строки, название товара, цена и количество. Добавьте функциональность для сортировки по каждому столбцу в возрастающем и убывающем порядке. При этом нумерация строк должна оставаться правильной, независимо от типа сортировки. Также необходимо выделять товары с количеством менее трех желтым цветом и товары с нулевым количеством красным цветом.

Реализуйте функцию подсчета общего количества товаров и их общей стоимости. Выведите эти значения внизу таблицы или в отдельном блоке.
