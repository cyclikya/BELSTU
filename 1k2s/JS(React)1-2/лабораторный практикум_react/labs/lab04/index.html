Лабораторная работа №4
Задание 1

Создайте компонент ToDoList, включающий в себя компоненты ToDoForm и ToDoItems. Он должен позволять пользователю добавлять задачи в список и отображать список задач с возможностью отметить их выполнение.

Компонент ToDoForm должен содержать форму для добавления новой задачи. Пользователь должен иметь возможность ввести название задачи и нажать кнопку Add, чтобы добавить ее в список.

Компонент "ToDoItems" должен отображать список задач. Каждая задача должна содержать название и флажок, позволяющий отметить ее выполнение.

Добавить кнопку Submit при нажатии на которую появляется список заданий, при выделении "чекбокса" и повторном нажатии на кнокпу задания должны обновится в соответствии с выбором.

Задание 2

Добавьте функциональность фильтрации задач. Пользователь должен иметь возможность отображать только выполненные или только невыполненные задачи.
